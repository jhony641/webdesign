DEFINICION

La semántica en HTML se refiere al uso de etiquetas que tienen un significado claro y específico sobre el contenido que contienen. Estas etiquetas no solo organizan la información visualmente, sino que también ayudan a los motores de búsqueda y a las tecnologías de asistencia a interpretar el propósito de cada elemento en una página web.

BENEFICIOS

los beneficios de usar etiquetas semanticas incluyen:

  • Mejora del SEO (Optimización para Motores de Búsqueda): Las etiquetas semánticas ayudan a los motores de búsqueda a entender mejor el contenido de una página, lo que puede mejorar su clasificación en los resultados de búsqueda.
  • Accesibilidad: Facilita la navegación para usuarios con discapacidades, ya que las tecnologías de asistencia, como los lectores de pantalla, pueden interpretar correctamente el propósito de cada sección.
  • Legibilidad del código: El uso de etiquetas semánticas hace que el código sea más fácil de leer y mantener, tanto para los desarrolladores actuales como para futuros colaboradores.
  • Estandarización: Promueve buenas prácticas de desarrollo al seguir estándares establecidos, lo que resulta en un código más limpio y estructurado.
  • Compatibilidad futura: Las etiquetas semánticas están diseñadas para ser compatibles con futuras tecnologías y navegadores, asegurando que el contenido siga siendo relevante y funcional.

ELEMENTOS

  • <header>define un encabezado
  • <nav>define conjunto de enlaces de navegación
  • <main>indica el contenido principal de un sitio
  • <aside>contenido relacionado
  • <footer>el pie de una seccion o pagina
  • <section>representa una secciongenerica
  • <article>representa una continuidad independiente
  • <figure>agrupa contenido
