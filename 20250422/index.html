CURSO DE DISEÑO WEB

HTML 5

HTML5 es la quinta versión del lenguaje de marcado HTML (HyperText Markup Language) y representa una evolución significativa en el desarrollo de páginas web. Se diseñó para mejorar la accesibilidad, la interactividad y la compatibilidad con dispositivos modernos.

Algunas de sus características clave incluyen:

  • Soporte para multimedia: Permite la integración nativa de audio y video sin necesidad de complementos como Flash.
  • Elementos semánticos: Introduce etiquetas como article, section y header que mejoran la estructura y comprensión del contenido.
  • Mejora en la compatibilidad con dispositivos móviles: Facilita el diseño adaptable y la optimización para diferentes tamaños de pantalla.
  • Capacidades avanzadas de almacenamiento y gráficos: Incluye tecnologías como Web Storage, Canvas y SVG para gráficos más dinámicos.

CSS

CSS (Cascading Style Sheets) es un lenguaje de estilo utilizado para definir la apariencia y el diseño de documentos HTML. Permite controlar aspectos visuales como colores, fuentes, márgenes, espaciado y posiciones de los elementos en una página web.

Algunas características clave de CSS:

  • Separación entre contenido y presentación: Mantiene el código HTML limpio y estructurado al definir los estilos en archivos separados.
  • Herencia y cascada: Permite aplicar estilos de manera jerárquica, evitando redundancias en el código.
  • Diseño adaptable: Facilita la creación de interfaces responsivas para diferentes dispositivos y tamaños de pantalla.
  • Animaciones y efectos: Permite agregar transiciones, efectos visuales y animaciones sin necesidad de JavaScript.

JavaScript

JavaScript es un lenguaje de programación interpretado que permite agregar interactividad y dinamismo a las páginas web. Es uno de los pilares del desarrollo web junto con HTML y CSS, y se ejecuta en el navegador del usuario.

Algunas de sus características clave:

  • Interactividad: Permite responder a eventos del usuario, como clics, movimientos del mouse o entrada de datos.
  • Manipulación del DOM:Puede modificar la estructura y el contenido de una página web en tiempo real.
  • Programación asincrónica:Facilita la carga de datos sin recargar la página mediante funciones como fetch() y async/await.
  • Amplia compatibilidad:Se ejecuta en la mayoría de los navegadores sin necesidad de instalación adicional.
  • Uso en el backend:Gracias a entornos como Node.js, también se emplea para la programación del servidor.
