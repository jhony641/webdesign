GRID-TEMPLATE-COLUMNS-ROWS

When working with CSS Grid, grid-template-columns and grid-template-rows are fundamental properties that allow you to define the structure of your grid by specifying the number and size of your columns and rows.

GRID-TEMPLATE.AREAS

grid-template-areas is a powerful CSS Grid property that allows you to define the layout of your grid by naming specific areas within it and then visually arranging those names to create your desired structure. It's often used for main page layouts because it makes the grid's structure very readable and easy to understand directly in the CSS

COLUMN-GAP,ROW-GAP,GAP

In CSS Grid (and also in Flexbox and multi-column layouts), column-gap, row-gap, and the gap shorthand property are used to define the spacing (often called "gutters") between grid tracks (columns and rows). These properties are applied to the grid container. They create space between grid items, not around the outer edges of the grid

GRID-JUSTIFY-CONTENT

justify-content only has a visible effect when the total width of your defined grid columns (grid-template-columns) is less than the available width of the grid container. If your columns fill the entire container (e.g., using 1fr units that naturally expand), justify-content won't have any extra space to distribute

CSS GRID

Es un sistema de diseño de layouts en CSS que permite crear estructuras de cuadrícula complejas y flexibles para organizar contenido en una página web. Con CSS Grid, puedes definir una cuadrícula de filas y columnas y colocar elementos en ella de manera precisa y flexible.

Es una herramienta poderosa para crear layouts complejos y lexibles en la web. Con su facilidad de uso y flexibilidad, es una excelente opción para diseñadores y desarrolladores que buscan crear sitios web modernos y responsivos. Al entender los conceptos básicos y las propiedades de CSS Grid, puedes crear diseños de layouts complejos y atractivos que se adapten a diferentes tamaños de pantalla y dispositivos.
